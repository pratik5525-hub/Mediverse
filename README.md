Mediverse is a medical report storing app which is more than just a storage app. This app can be used to share the medical report without revealing identity. And this app work as cloud storage so user can access their report from anywhere. This app also have emergency card option which allow authorized hospital to access user's crtical medical information in case of emergency. This app also have a in built chatbot which advice user and also this ai analyse the report and use the data fro health tracking.
